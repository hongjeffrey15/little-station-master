## Mulberry Symbols

The literal vocabulary illustrations in `assets/pics/` are from
**[Mulberry Symbols](https://mulberrysymbols.org/)**, copyright © Steve Lee,
licensed under the
[Creative Commons Attribution-ShareAlike licence](https://creativecommons.org/licenses/by-sa/2.0/uk/)
(CC BY-SA 2.0 UK). Source: https://github.com/mulberrysymbols/mulberry-symbols
(see `MULBERRY-LICENSE.txt`). Files are renamed but otherwise unmodified;
any future modified versions remain under the same licence.

Mulberry Symbols are an AAC (Augmentative and Alternative Communication)
symbol set designed so that each picture literally depicts its word — chosen
here because young learners with limited vocabulary need the illustration to
*be* the word, not a visual pun.

## Emoji

Where no honest symbol exists (e.g. ✋ hand, 👃 nose, 🌊 sea) the app uses the
device's native emoji font, which requires no attribution.

## Hong Kong icon pack

HK-specific items with no equivalent in any symbol set — 蝦餃 (hargow),
燒賣 (siumai), 蛋撻 (eggtart), 奶茶 (milktea), 電車 (tram), 利是 (laisee),
月餅 (mooncake), 燈籠 (lantern) — are original flat SVG icons drawn for this
project in a style matching Mulberry Symbols. They are released under this
repository's own licence.
